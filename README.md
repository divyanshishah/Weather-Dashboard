# 🌤 Weather Dashboard

A simple and responsive Weather Dashboard built using HTML, CSS, and JavaScript.

This application fetches real-time weather information using asynchronous JavaScript and displays weather details such as temperature, humidity, wind speed, and precipitation based on the city entered by the user.

---

## 📸 Project Preview

Your dashboard includes:

* 🌡 Temperature
* 💧 Humidity
* 🌬 Wind Speed
* ☔ Precipitation
* 🔍 City Search Feature

---

## 🚀 Features

✔ Search weather by city name
✔ Real-time weather data
✔ Async/Await with Fetch API
✔ REST API integration
✔ Dynamic DOM updates
✔ Error handling
✔ Responsive UI design
